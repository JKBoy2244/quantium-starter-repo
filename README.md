# Quantium starter repo
This repo contains everything you need to get started on the program! Good luck!

# How to Run Each Task

- To run task 2, type exactly on terminal: python task_2/data_merge.py 

- To run task 3, type exactly on terminal: python task_3/app.py

- To run task 4, type exactly on terminal: python task_4/appStyled.py

- To run task 5, type exactly on terminal: python task_5/test_app.py

- To run task 6, type exactly on terminal: sh task_6/execute_tests.sh

-(For each, after typing, press enter).
